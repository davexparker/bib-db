Below are some examples of the ways you can change the style of your lists using CSS.

Look at the CSS file to see how it is done.

First some in list format...

  • list.php?css_file=list1.css
  • list.php?css_file=list2.css

Then some in table format...

  • table.php?css_file=table1.css
  • table.php?css_file=table2.css
  • table.php?css_file=table3.css

Note that for each set, the PHP/HTML code is the same; just the CSS changes.
